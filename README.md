## 介绍 
一款优化 `MIUI` 系统中各种不友好功能的模块

## 功能列表
- [x] 阻止结束特定软件的进程
- [x] 解除系统应用不允许关闭通知的限制
- [x] 修复传送门无法正常打开链接的问题
- [x] 隐藏应用商店中杂乱的赞助商应用推荐
- [x] 使手机在未解锁的状态下依然能够识别NFC卡片
- [x] 禁用应用安装器的安全检测
- [x] 取消设置中二次确认对话框的倒计时
- [x] 禁用手机管家中不重要的扫描项
- [x] 修复网络助手的校正总是失败的问题
- [x] 移除开发者选项中ADB相关设置的开关限制
- [x] 清除设置中的系统新版本标记
- [x] 禁止超级壁纸暂停
- [x] 移除多个系统应用的开屏广告
- [x] 编辑指纹快捷功能菜单项
- [x] 使软件的启动开屏画面正常显示
