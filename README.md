## 介绍 
一款优化 `MIUI` 系统中各种不友好功能的模块

## 功能列表
- [x] 去除若干开屏广告+软件内广告
- [x] 阻止结束软件进程
- [x] 解除关闭通知限制
- [x] 修复传送门相关bug
- [x] 系统桌面美化
- [x] 应用商店功能优化
- [x] MIUI+优化
- [x] NFC相关优化
- [x] 禁用应用安装检测
- [x] 解除ADB相关限制
- [x] 手机管家优化
- [x] 修复网络助手相关bug
- [x] 清除系统新版本标记
- [x] 超级壁纸优化
- [x] 编辑指纹快捷功能
- [x] 隐藏状态栏图标
- [x] 系统界面美化
